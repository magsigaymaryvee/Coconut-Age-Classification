# Coconut-Age-Classification
A Streamlit-based web and mobile application that classifies coconut tree age using trunk images and height approximation. The system uses a MobileNet CNN model to predict age groups: Young (5–20 years), Pre-Mature (21–40 years), and Mature (41–60 years). Built with Python, TensorFlow, Keras, NumPy, Pandas, and PIL.
